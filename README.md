# Cashflow Harian

Front-end statis untuk aplikasi keuangan harian. Dibangun hanya dengan HTML, TailwindCSS Play CDN, dan JavaScript native.

## Menjalankan Aplikasi

Karena halaman dan komponen dimuat dengan `fetch()`, jalankan melalui server statis:

```bash
python3 -m http.server 8080
```

Lalu buka:

```txt
http://localhost:8080
```

## Struktur Utama

- `index.html` sebagai shell aplikasi.
- `components/` untuk sidebar, topbar, dan mobile navigation.
- `pages/` untuk konten halaman yang dimuat ke `#app-content`.
- `assets/js/data.js` untuk data dummy dan state lokal.
- `assets/js/menu.js` untuk menu dinamis.
- `assets/js/app.js` untuk routing, render UI, validasi form, modal, toast, dan simulasi CRUD.
